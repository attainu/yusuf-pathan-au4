Facebook.com

Friends

"Friending" someone on the platform is the act of sending another user a "friend request" on Facebook. The two people are Facebook friends once the receiving party accepts the friend request. In addition to accepting the request, the user has the option of declining the friend request or hiding it using the "Not Now" feature.

Wall

The Wall is the original profile space where Facebook users' content until December 2011 was displayed. It allowed the posting of messages, often short or temporal notes, for the user to see while displaying the time and date the message was written

Notifications

Notifications tell the user that something has been added to his or her profile page. Examples include: a message being shared on the user's wall or a comment on a picture of the user or on a picture that the user has previously commented on

Events

Facebook events are a way for members to let friends know about upcoming events in their community and to organize social gatherings. Events require an event name, network, host name, event type, start time, location, and a guest list of friends invited
